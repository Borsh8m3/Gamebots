# 🎮 GameBots: A Pygame Environment for Training and Comparing Supervised Learning & Reinforcement Learning (PPO) Agents

> 📄 Check out my full project report on my website: [https://bit.ly/game_bots](https://bit.ly/game_bots)

A **BBTAN** clone built in Python (Pygame) as an AI sandbox. This project explores and compares different Artificial Intelligence approaches: from pure math (Aimbot) and Machine Learning (Random Forest), all the way to advanced Reinforcement Learning (PPO).

Welcome to the **GameBots** project! The goal of this repository is to build a clone of the popular mobile game **BBTAN** from scratch using Pygame, and then explore and compare different AI approaches that learn to play it.

From random shots and hard math to advanced neural networks — this project is a complete testing laboratory for AI algorithms.

---

## 🧠 Tested Bots

Four distinct "brains" were implemented and tested within the project:

### 1. 🎲 Random Bot (Baseline)
The simplest possible agent. Chooses a completely random shooting angle every turn. It serves as a baseline to evaluate whether more advanced models are actually learning anything. On average, it loses around level 10.

### 2. 🎯 Math Aimbot (The Teacher)
An algorithm based on pure math and geometry. It computes perfect trajectories and ricochets, playing flawlessly. It acts as a **"teacher"** by generating ideal training data — hundreds of thousands of rows containing board state information and optimal shot angles.

### 3. 📚 Supervised Learning Agent (The Student)
A supervised learning model (e.g. `RandomForestRegressor` / `RandomForestClassifier`) trained on the massive dataset generated by the Aimbot.

- **Results:** The bot quickly picks up the basics and can regularly reach around **level 24**.
- **Findings:** Reveals the classic "glass ceiling" of regression algorithms. In dense, critical situations, the model averages its decisions (instead of committing to a risky ricochet), which leads to an inevitable loss around level 30.

### 4. 🏋️ Reinforcement Learning Agent (PPO — The Master)
True artificial intelligence based on Reinforcement Learning, using the **PPO (Proximal Policy Optimization)** algorithm from the `stable-baselines3` library.

- The model does **not** rely on pre-built math formulas — it learns the game's physics on its own through trial and error, maximizing rewards for destroying blocks and surviving.
- **Results:** After sufficient training (65k+ steps), the agent can survive over **600 levels**, effectively keeping blocks away from the bottom edge of the screen (maintaining them at an average height of floor 9 out of 10).

## 📊 Data Analysis & Training

The project places strong emphasis on analytics and tracking AI progress. The repository includes scripts for data analysis using the `pandas` library.

- **TensorBoard Exploration:** PPO agent training is monitored in real time, allowing observation of classic learning curves — such as an early performance dip during exploration, followed by a steady rise in mean reward and agent lifespan.
- **Effectiveness Histograms:** Visualizations of the SL bot's "glass ceiling", proving the limitations of supervised learning in environments that demand precise, split-degree decisions.

## 🚀 Getting Started

**1. Clone the repository:**
```bash
git clone https://github.com/Borsh8m3/Gamebots.git
cd Gamebots
```

**2. Install the required libraries:**
```bash
pip install -r requirements.txt
```

**3. Launch the project:**

The main entry point is the `Main.ipynb` notebook.

---

## 🕹️ Manual Mode (Play it yourself!)

You can test the game physics yourself. Simply run the main game file, click and hold the left mouse button, aim, and release:

```bash
python game.py
```
